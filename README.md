# vue3-project

## vue3 Update Guide
```
// 전역 @vue/cli 버전 체크 후 4.4.x 로 설치
npm install -g @vue/cli

// 프로젝트 설치
vue create yourProject
cd /yourProject

//베타 버전 설치
vue add vue-next
//
```
